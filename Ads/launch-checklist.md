# Paid Ads Launch Checklist

## Tracking

- Confirm `Start Free Beta` is tracked as a primary conversion
- Confirm `Book a 15-Minute Workflow Review` is tracked as a secondary conversion
- Confirm landing-page views are visible in analytics
- Confirm Google Ads receives conversion data
- Confirm UTM parameters are preserved through signup flow

Recommended UTM set:

```text
utm_source=google
utm_medium=cpc
utm_campaign=dealerwyze_os_automation_hook
utm_term={keyword}
utm_content={creative}
```

## Google Ads Setup

- Create campaign `US | Search | DealerWyze OS | Automation Hook`
- Use Search only
- Set budget to `$25/day`
- Start with `Maximize Clicks`
- Add all shared negative keywords
- Launch first 3 ad groups only:
  - `Dealer Communication Automation`
  - `Independent Dealer CRM`
  - `Dealership Management Software`

## Landing Page

- Use `/lp/dealerwyze-os`
- Verify page is public without login
- Verify CTA buttons go to the intended destinations
- Verify mobile layout and hero CTA visibility

## Product / Onboarding Notes

- Make sure starter sequences are easy to find after signup
- Make sure Gmail connection is obvious during onboarding
- Make sure voice and social setup are clearly marked as one-time setup steps
- Make sure customer edit flows support birthday and service data entry

## Week 1 Review

- Check search terms daily
- Add negatives aggressively
- Pause irrelevant traffic fast
- Compare beta-start quality by ad group
- Watch CTR and landing-page conversion rate

## Week 2 Review

- Shift spend to the best-converting ad groups
- Pause expensive low-quality keywords
- Consider launching the BHPH ad group if search intent looks strong
- Consider retargeting only after meaningful traffic exists
