# Ads

This folder documents the current paid acquisition work for DealerWyze.

Current primary landing page:
- `/lp/dealerwyze-os`

Current strategy:
- Start with Google Search
- Lead with the automation / dealership operating system angle
- Use the landing page to prove DealerWyze is more than a CRM

Files:
- `strategy.md`: positioning, audience, offer, and campaign direction
- `google-search-campaign.md`: the current Google Ads campaign structure and copy
- `landing-pages.md`: summary of the paid landing pages and recommended usage
- `launch-checklist.md`: tracking, setup, and launch steps
- `index.html`: browser-friendly summary of the ad program
- `google-search-campaign.html`: browser-friendly version of the current ad set

Recommended source of truth:
- Strategy decisions live in this folder
- The live page implementation lives in `app/lp` and `components/landing`
