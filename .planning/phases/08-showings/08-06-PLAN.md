2. Query all upcoming showings for the org (not filtered by listing_id):
   ```typescript
   const { data } = await supabase
     .from('showings')
     .select(`
       id, scheduled_at, status, org_id, listing_id,
       listing:vehicles(id, address),
       contact:customers(id, name, primary_phone),
       agent:profiles(id, full_name)
     `)
     .eq('org_id', profile.org_id)
     .eq('status', 'scheduled')

     .limit(200)
   ```
