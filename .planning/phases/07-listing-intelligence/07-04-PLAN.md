    Before applying the update, check if `price` is in the incoming fields AND differs from the current vehicle price. Scope this to RE listings by checking `currentVehicle.make === 'RE'` (the placeholder set by import-mls) — do not track price changes on dealer vehicle records.

    ```typescript
    // Inside PATCH handler, after fetching current vehicle and before building update payload
    const incomingPrice = body.price  // from validated fields
    if (
      currentVehicle.make === 'RE' &&
      incomingPrice !== undefined &&
      incomingPrice !== currentVehicle.price
    ) {
      const changeEntry = {
        from: currentVehicle.price,
        to: incomingPrice,
        changed_at: new Date().toISOString(),
      }
      updatePayload.price_change_log = [
        ...(Array.isArray(currentVehicle.price_change_log) ? currentVehicle.price_change_log : []),
        changeEntry,
      ]
      updatePayload.price_change_count = (currentVehicle.price_change_count ?? 0) + 1
    }
    ```
