<task type="auto">
  <name>Task 2: Commission Plans CRUD routes</name>
  <files>app/api/commission-plans/route.ts, app/api/commission-plans/[id]/route.ts</files>
  <action>
    Create `app/api/commission-plans/route.ts`:

    **GET** — list all plans for the org:
    1. `requireProfile()`
    2. Verify RE org (organizations.vertical = 'real_estate'). 403 if dealer.
    3. Query `commission_plans WHERE org_id = profile.org_id ORDER BY is_default DESC, created_at ASC LIMIT 100`.
    4. Return 200 `{ plans: rows }`.

    **POST** — create a new plan (broker/admin only):
    1. `requireProfile()`
    2. Verify RE org. 403 if dealer.
    3. `isDealerAdmin(profile.role)` from `lib/auth/dealerRoles.ts`. Return 403 `{ error: 'Forbidden' }` if false.
    4. Parse body with `CommissionPlanCreateSchema`. Return 400 on failure.
    5. If `is_default = true` and `agent_id = null`: check if a default already exists for this org. If yes, UPDATE existing default to set `is_default = false` first (swap default pattern — only one default per org).
    6. Compute `broker_split_pct = 100 - agent_split_pct` if not provided and plan_type = 'percentage_split'.
    7. INSERT into commission_plans with `org_id = profile.org_id`. Return 201 with created row.

    Create `app/api/commission-plans/[id]/route.ts`:

    **PATCH** — update a plan (broker/admin only):
    1. `requireProfile()`
    2. Verify RE org. 403 if dealer.
    3. `isDealerAdmin(profile.role)`. Return 403 if false.
    4. Parse body with `CommissionPlanUpdateSchema`. Return 400 on failure.
    5. Verify plan belongs to org: query `commission_plans WHERE id = params.id AND org_id = profile.org_id`. Return 404 if not found.
    6. If update sets `is_default = true`: clear any other org default first (same swap as POST).
    7. UPDATE the plan. Return 200 with updated row.

    **DELETE** — delete a plan (broker/admin only):
    1. `requireProfile()`
    2. Verify RE org. 403 if dealer.
    3. `isDealerAdmin(profile.role)`. Return 403 if false.
    4. Verify plan belongs to org. Return 404 if not found.
    5. Check if any open transactions reference this plan via `commission_plan_id`. If yes, return 409 `{ error: 'Plan is in use by open transactions' }`.
    6. DELETE the plan. Return 200 `{ deleted: true }`.

    Across both files: `createClient()`, never `createServiceClient()`. No stack traces in error responses. Use `{ error: 'Internal error' }` on 500.
  </action>
  <verify>
    POST without auth: 401. POST as agent role: 403. POST with admin auth and valid body: 201.
    PATCH to change is_default: previous default becomes false. DELETE on plan in use: 409.
    `npx tsc --noEmit` passes.
  </verify>
  <done>Four handlers: GET (any RE org member), POST/PATCH/DELETE (admin/broker only). Default plan swap logic correct. Plan-in-use guard on delete. All tenant-isolated.</done>
