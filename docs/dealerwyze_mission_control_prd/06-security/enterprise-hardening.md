# Enterprise Hardening

## Security Philosophy

Security must be foundational, not retrofitted.

## Requirements

- strict tenant isolation
- least privilege access
- encrypted secrets
- audit logging
- immutable event history
- role-based permissions
- session monitoring
- API rate limiting

## AI Security

- prompt injection protection
- tenant context isolation
- scoped tool permissions
- moderation pipelines
- approval workflows

## Infrastructure Security

- Cloudflare WAF
- container isolation
- secret rotation
- backup encryption
- monitoring
- alerting

## Compliance

Must support:
- TCPA workflows
- audit exports
- dealership record retention
- consent tracking
