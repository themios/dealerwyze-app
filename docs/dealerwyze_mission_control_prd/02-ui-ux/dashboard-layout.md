# Dashboard Layout

## Layout Philosophy

Mission Control must prioritize:
- speed
- visibility
- operational awareness
- drill-down depth

## Global Header

Persistent metrics:
- active leads
- overdue follow-ups
- appointment rate
- inventory aging
- system health
- AI queue load
- delinquency alerts
- revenue KPIs

## Main Grid Layout

Grid-based operational modules:
- Lead Radar
- Communications
- Inventory
- BHPH
- Marketing
- AI Agents
- Analytics
- Mission Feed

## Mission Feed

Realtime chronological activity stream.

Event cards include:
- severity
- source
- actor
- timestamp
- linked entities
- AI summary
- action buttons

## Interaction Model

Every widget must:
- support drill-down
- support filtering
- support quick actions
- support contextual AI actions

No dead widgets allowed.

## Mobile Strategy

Mobile design priorities:
- operational triage
- communication workflows
- urgent alerts
- quick approvals
- appointment management
