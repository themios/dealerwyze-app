# Messages & Email

## What is the Messages page?

The **Messages** page is where you manage all communication with your leads and customers:
- Incoming messages (SMS, email)
- Outgoing messages
- Message threads
- Email conversations

Everything is organized by thread so you can see the full conversation history with each person.

---

## Accessing Messages

Click **Messages** in the left sidebar.

---

## Page Layout

### Left Pane (Desktop & Mobile)
- **Search box** — Find messages by subject, name, or content
- **Thread list** — Shows all conversation threads
  - Type badge (SMS, Email, etc.)
  - Status badge (New, Replied, etc.)
  - Subject line
  - Last message preview
  - Unread indicator (red badge shows count)

Click any thread to open it.

### Right Pane (Desktop Only; Mobile shows full screen)
- **Back button** (mobile only) — Return to thread list
- **Thread header** — Subject line + type/status badges
- **Channel tabs** (2 tabs):
  - **Messages** (speech bubble icon) — SMS/in-app messages in this thread
  - **Emails** (envelope icon) — Email messages in this thread
  - Each tab shows total count and unread count
- **Message list** — Scrollable history of all messages (older above, newer below)
  - Timestamps
  - Attachments
  - Read/unread status
- **Composer** (fixed at bottom):
  - Text area — Type your message
  - **Paperclip button** — Attach files (opens file picker)
  - **Send button** — Changes text:
    - "Send Message" (for SMS)
    - "Send Email" (for email)
  - File preview area — Shows attached files with remove (X) buttons
  - **Keyboard shortcut:** Cmd/Ctrl+Enter sends

---

## How to Send a Message

1. Open a thread or start a new conversation
2. Type your message in the **Composer** text area
3. (Optional) **Add attachments:**
   - Click the **Paperclip icon**
   - Select file from your computer
   - File appears in preview area below composer
4. Click **Send Message** or **Send Email**
5. Message appears in thread immediately

---

## How to Reply to a Message

1. Open the thread (click on it in the thread list)
2. Read the full conversation history
3. Scroll to the bottom
4. Type your reply in the **Composer**
5. Click **Send**

Your reply is added to the conversation and the other person is notified.

---

## Message Types

### SMS (Text Messages)
- Short messages (160 characters)
- Instant delivery
- Shows as "SMS" in thread badges
- Tab label: **Messages**

### Email
- Full-length messages
- Supports attachments
- Shows as "Email" in thread badges
- Tab label: **Emails**

Switch between **Messages** and **Emails** tabs to see both types of communication in the same thread.

---

## Thread Status Badges

- **New** — Unread message in this thread
- **Replied** — You've responded
- **Pending** — Waiting for response
- **Resolved** — Thread marked complete

---

## Tips

- **Check unread counts** — The red badge on thread list shows how many unread messages
- **Use search** — Quickly find old conversations by name or keyword
- **Keep threads organized** — Reply promptly so threads don't pile up
- **Attach files** — Share documents, photos, or contracts directly in messages
- **See channel history** — Use the **Messages** and **Emails** tabs to see different communication types
